# Conversor de Unidades ISCAT

Este projeto consiste num **Conversor de Unidades** desenvolvido em **Python** utilizando a biblioteca **Tkinter** para a interface gráfica de utilizador (GUI). Foi criado como parte dos requisitos da disciplina de Programação I do Instituto Superior Politécnico da Catepa (ISCAT), curso de Engenharia Eletrotécnica.

## Funcionalidades

O programa permite a conversão entre diversas unidades nas seguintes categorias:

- **Comprimento:** Quilômetro (km), Metro (m), Centímetro (cm)

- **Massa:** Quilograma (kg), Grama (g)

- **Temperatura:** Celsius (°C), Fahrenheit (°F)

A interface é intuitiva, permitindo ao utilizador selecionar a categoria, inserir o valor e escolher as unidades de origem e destino para realizar a conversão.

## Requisitos

Para executar este programa, é necessário ter o Python instalado na sua máquina. A biblioteca `tkinter` geralmente vem pré-instalada com o Python. Caso não tenha, pode instalá-la ou garantir que o seu ambiente Python a inclui.

- **Python 3.x**

- **Tkinter** (geralmente incluído com a instalação padrão do Python)

Após descarregar ou clonar o repositório, navegue até à pasta do projeto e execute o ficheiro `conversor.py` usando o interpretador Python:

```bash

```

## Utilização

1. **Selecione a Categoria:** Escolha entre "Comprimento", "Massa" ou "Temperatura" na lista suspensa.

1. **Insira o Valor:** Digite o número que deseja converter no campo "Valor:".

1. **Selecione as Unidades:** Escolha a unidade de origem ("De:" ) e a unidade de destino ("Para:") nas respetivas listas suspensas.

1. **Converta:** Clique no botão "CALCULAR CONVERSÃO".

1. **Visualize o Resultado:** O resultado da conversão será exibido abaixo do botão.
